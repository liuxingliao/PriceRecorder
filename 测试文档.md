- **项目名称**: 商品价格记录

## 1. 测试概述

### 1.1 测试目标
- 验证应用功能符合需求文档要求
- 确保数据模型正确持久化
- 验证用户界面交互流畅
- 测试边界条件和异常情况
- 确保暗黑模式正常工作

### 1.2 测试范围
- 单元测试：数据模型、服务层
- UI测试：主要界面导航和交互
- 手动测试：完整功能流程

### 1.3 测试环境
- **开发环境**: Xcode 15.0+
- **目标系统**: iOS 17.0+
- **测试设备**: iPhone模拟器 / 真机
- **数据存储**: 内存数据库（测试）/ SwiftData（生产）

- [ ] 能够在模拟器中启动
- [ ] 能够在真机中安装运行
- [ ] 单元测试能够编译通过
- [ ] UI测试能够编译通过

### 4.2 编译命令

```bash
# Debug 编译
xcodebuild -scheme PriceRecorder -configuration Debug build

# Release 编译
xcodebuild -scheme PriceRecorder -configuration Release build

# 清理并编译
xcodebuild clean build -scheme PriceRecorder
```

### 4.3 常见编译问题

| 问题 | 可能原因 | 解决方案 |
|-----|---------|---------|
| SwiftData 相关错误 | iOS版本过低 | 确保部署目标 ≥ iOS 17.0 |
| Bundle Identifier 错误 | 未配置开发团队 | 在Xcode中配置Team |
| 权限描述缺失 | Info.plist配置 | 检查相机/相册权限描述 |
| 模块未找到 | 文件引用问题 | 检查文件是否正确添加到target |

## 7. 缺陷严重级别定义

| 级别 | 描述 | 示例 |
|-----|------|-----|
| P0 - 严重 | 应用崩溃、核心功能不可用 | 无法保存商品、应用启动崩溃 |
| P1 - 高 | 主要功能异常、数据丢失 | 搜索不工作、CSV导出失败 |
| P2 - 中 | 次要功能问题、UI问题 | 排序异常、暗黑模式颜色错误 |
| P3 - 低 | 优化建议、体验问题 | 文字对齐、动画不流畅 |
