## 1. 项目概述

### 1.1 项目背景
开发一款iOS应用，帮助用户记录不同商家的商品价格信息。用户可通过豆包智能录入快速录入商品，并可随时查看历史价格、比较不同商家的价格。

### 1.2 核心功能
- 豆包智能录入 + 手动输入商品
- 商家管理（支持分类）
- 商品搜索和比价
- 历史价格趋势图
- iCloud备份和CSV导入导出
- 数据统计

## 2. 功能需求详细说明

### 2.1 首页 - 最近录入

**功能描述**: 显示用户最近录入的商品记录

**详细需求**:
- 显示最近录入的10条商品记录
- 每条记录一行，显示：商品名称、价格、数量、单位
- 点击商品进入详情页
- 页面顶部显示"商品录入"按钮
- 点击"商品录入"按钮弹出选择框：手动输入 / 豆包录入
- 空状态时显示提示信息和快速录入入口

---

### 2.2 商品录入功能

#### 2.2.1 录入方式选择
- 提供两种录入方式：手动输入、豆包录入
- 用户可自由切换选择

#### 2.2.2 豆包录入流程
1. **打开豆包**: 点击按钮打开配置的豆包链接
2. **复制数据**: 在豆包中复制商品JSON数据
3. **粘贴JSON**: 将JSON数据粘贴到编辑框
4. **解析商品**: 点击解析按钮，解析JSON数据
5. **确认编辑**:
   - 显示解析出的商品列表
   - 支持删除商品
   - 支持编辑商品详情
   - 点击商品进入详情编辑页
6. **保存确认**:
   - 选择商家（支持新增商家）
   - 确认/修改购买时间（默认当前时间）
   - 保存

#### 2.2.3 手动输入流程
- 支持一次输入多个商品
- 流程与豆包录入的编辑页类似
- 可随时添加/删除商品

#### 2.2.4 JSON数据格式
```json
[
  {
    "name": "商品名称",
    "quantity": 1.0,
    "unit": "个",
    "totalPrice": 10.0,
    "spec": "规格（可选）"
  }
]
```

---

### 2.3 搜索页面

**功能描述**: 搜索商品记录，支持多种查看方式

#### 2.3.1 搜索模式切换
- 顶部搜索框 + 切换按钮
- 两种模式：商品模式 / 商家模式

#### 2.3.2 商品模式
- 每行显示：商品名称、商家、时间、总价、数量、单价
- 点击进入商品详情页
- 支持修改数据

#### 2.3.3 商家模式
- 每行显示：商家名称、该商品最近一次价格、时间、总价、数量、单价
- 点击显示该商家该商品的所有历史记录

#### 2.3.4 排序功能
支持以下排序方式：
- 按时间
- 按价格
- 按商品/商家名称首字母

---

### 2.4 比价功能

**功能描述**: 比较同一商品在不同商家的价格

#### 2.4.1 比价流程
1. **选择商品**: 从商品列表中选择一个商品
2. **选择商家**: 最多选择5个商家进行比较
3. **显示结果**:
   - 历史价格趋势图（显示最近100次数据）
   - 最新价格对比表
   - 标识出最便宜的商家

---

### 2.5 商品详情页

**功能描述**: 显示商品的完整信息并支持编辑

**显示字段**:
- 商品名称
- 品牌
- 规格
- 数量、单位
- 总价、单价
- 商家
- 购买时间
- 小票照片（如有）
- 备注

**功能**:
- 支持编辑所有字段
- 支持添加/更换小票照片
- 单价修改后不更新总价

---

### 2.6 设置页面

#### 2.6.1 商家管理
- 商家列表显示
- 新增/编辑/删除商家
- 商家信息：名称、分类、地址、电话、备注
- 支持商家分类管理（用户自由添加分类）

#### 2.6.2 豆包配置
- 豆包链接配置
- 支持自定义豆包链接

#### 2.6.3 数据导入导出
- **CSV导出**: 导出全部数据，一行一条，包含所有字段
- **CSV导入**: 仅支持本应用导出的CSV格式，支持商家去重
- 导入结果显示（成功/失败数量）

#### 2.6.4 iCloud备份
- 手动备份功能
- 自动备份开关
- 显示上次同步时间
- 全量备份

#### 2.6.5 数据统计
- 商品总数
- 商家数量
- 分类数量
- 总支出金额
- 最贵商品
- 最常购买商品
- 商家商品数量排行（前5）

#### 2.6.6 调试功能
- 清空所有数据按钮（需二次确认）

## 3. 数据模型设计

### 3.1 商品记录 (ProductRecord)
| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| id | UUID | 是 | 唯一标识 |
| name | String | 是 | 商品名称 |
| brand | String? | 否 | 品牌 |
| quantity | Double | 是 | 数量 |
| unit | String | 是 | 单位 |
| spec | String? | 否 | 规格 |
| totalPrice | Double | 是 | 总价 |
| unitPrice | Double | 是 | 单价（自动计算） |
| merchantID | UUID | 是 | 所属商家ID |
| purchaseDate | Date | 是 | 购买时间 |
| receiptPhoto | Data? | 否 | 小票照片 |
| notes | String? | 否 | 备注 |
| createTime | Date | 是 | 创建时间 |
| updateTime | Date | 是 | 更新时间 |

### 3.2 商家 (Merchant)
| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| id | UUID | 是 | 唯一标识 |
| name | String | 是 | 商家名称 |
| categoryID | UUID? | 否 | 分类ID |
| address | String? | 否 | 地址 |
| phone | String? | 否 | 电话 |
| notes | String? | 否 | 备注 |
| createTime | Date | 是 | 创建时间 |
| updateTime | Date | 是 | 更新时间 |

### 3.3 商家分类 (MerchantCategory)
| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| id | UUID | 是 | 唯一标识 |
| name | String | 是 | 分类名称 |
| createTime | Date | 是 | 创建时间 |

### 3.4 小票 (Receipt)
| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| id | UUID | 是 | 唯一标识 |
| merchantID | UUID | 是 | 商家ID |
| purchaseDate | Date | 是 | 购买时间 |
| photo | Data? | 否 | 照片 |
| notes | String? | 否 | 备注 |
| createTime | Date | 是 | 创建时间 |

### 3.5 API配置 (APIConfig)
| 字段 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| id | UUID | 是 | 唯一标识 |
| doubaoLink | String | 是 | 豆包链接 |

| updateTime | Date | 是 | 更新时间 |

## 4. 页面结构和导航流程

```
TabView (底部标签栏)
├── 首页 (HomeView)
│   └── 商品详情页 (ProductDetailView)
│
├── 搜索 (SearchView)
│   ├── 商品模式 / 商家模式
│   ├── 商品详情页 (ProductDetailView)
│   └── 比价页 (PriceComparisonView)
│
└── 设置 (SettingsView)
    ├── 商家管理 (MerchantManagementView)
    │   ├── 商家编辑 (MerchantEditView)
    │   └── 商家分类管理 (CategoryManagementView)
    ├── 豆包配置 (APIConfigView)
    └── 数据管理 (DataManagementView)
        ├── iCloud备份 (iCloudBackupView)
        ├── CSV导入导出 (CSVImportExportView)
        └── 数据统计 (StatisticsView)

商品录入流程
├── 选择录入方式 (手动/豆包)
├── 豆包JSON解析
├── 商品列表编辑 (ProductListEditView)
└── 商品详情编辑 (ProductEditView)
```

## 5. 技术要求

### 5.1 技术栈
- **语言**: Swift 5.0+
- **UI框架**: SwiftUI
- **最低系统版本**: iOS 17.0+
- **数据持久化**: SwiftData
- **iCloud同步**: CloudKit + SwiftData
- **图表**: SwiftCharts (iOS 16+)
- **暗黑模式**: 支持

### 5.2 权限要求
- 相机访问权限（拍照，保留配置但暂无功能）

```
商品名称,品牌,数量,单位,规格,总价,单价,商家名称,商家分类,购买时间,备注,创建时间,更新时间
```
